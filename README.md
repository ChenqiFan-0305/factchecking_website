# scientific_rate_website
This website is developed for Columbia Business School's Research.

## Model
- Link:
  - LinkId
  - Article External Link
  - Status : unprocessed, processed with scientific score, processed without scientific score
- Article:
  - Title
  - Topic(Foreign Key)
  - Text
  - Image
  - Source(Foreign Key)
  - Scientific Rate
- Similarity Rate:
  - SimRateId
  - Article 1 Title
  - Article 2 Title
  - Similarity Rate
- Topic:
  - Topic
- Source:
  - Source
